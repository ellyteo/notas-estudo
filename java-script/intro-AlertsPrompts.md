Alert→ Serve para exibir uma mensagem

prompt→ mostrar uma pergunta ou receber uma resposta do usuário. Toda resposta é texto (string)

---

### como guardar as respostas→ criar uma variável:

**let:** declara uma variável com escopo de bloco, que pode ser retribuída, mas não redeclarada no mesmo espaço. 

**const:** declara uma variável com escopo de bloco que não pode ser retribuída, apenas seu conteúdo interno pode mudar (se for objeto/array)
