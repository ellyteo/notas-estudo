## Como clonar o repositório

abrir um repositório ->
clicar em "<> code" ->
selecionar "SSH" ->
copiar ->
abrir o Git blash ->
digitar :

```bash
cd 'nome de uma pasta existente'
```
-> enter
-> digitar:
```bash
git clone (clicar botão direito e selecionar "paste")
```
-> enter 2x
-> digitar:
```bash
cd 'nome do repositório'
```
-> enter
-> digitar:
```bash
code .
```
-> enter
